```sql
-- –ü–µ—Ä–µ–≤–µ—Å—Ç–∏ –∏—Ö –Ω–∞ –Ω–æ–≤—ã–π SUBSCRIPTION –ø–ª–∞–Ω
-- (–≤—ã–ø–æ–ª–Ω–∏—Ç—å –ø–æ—Å–ª–µ —Å–æ–∑–¥–∞–Ω–∏—è –Ω–æ–≤–æ–≥–æ –¥–µ—Ñ–æ–ª—Ç–Ω–æ–≥–æ –ø–ª–∞–Ω–∞)
UPDATE billing_accounts 
SET subscription_plan_id = (SELECT id FROM subscription_plans WHERE is_default = true AND plan_type = 'SUBSCRIPTION')
WHERE subscription_plan_id = 8;
```

### –®–∞–≥ 4: –°–≤—è–∑–∞—Ç—å –ø–ª–∞–Ω—ã —Å –∞–≥–µ–Ω—Ç–∞–º–∏
```sql
-- –£–±–µ–¥–∏—Ç—å—Å—è —á—Ç–æ —É –¥–µ—Ñ–æ–ª—Ç–Ω–æ–≥–æ –ø–ª–∞–Ω–∞ –µ—Å—Ç—å –¥–æ—Å—Ç—É–ø —Ö–æ—Ç—è –±—ã –∫ –æ–¥–Ω–æ–º—É –∞–≥–µ–Ω—Ç—É
SELECT p.id, p.name, COUNT(pa.agent_id) as agent_count
FROM subscription_plans p
LEFT JOIN plan_agents pa ON p.id = pa.plan_id
```
